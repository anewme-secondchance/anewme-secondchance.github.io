ID Help After Release

📝 ID Recovery Checklist

🚨 If your prison ID has not arrived yet — DO NOT WAIT. You can take action immediately.

1️⃣ Go to Texas DPS Yourself

Even if they told you it would be mailed, you can apply directly.

Bring:

  • Release paperwork
  • Prison ID (even if expired)
  • Social Security card (if available)
  • Birth certificate (if available)

Find DPS locations:

Texas DPS Office Locator

Ask for a temporary paper ID while you wait.

2️⃣ If You Do Not Have a Social Security Card

Go to a Social Security office and request a replacement card (free).

Bring:

  • Release paperwork
  • Any identification you have
  • Birth certificate (if possible)

Find office:

Social Security Office Locator

3️⃣ If You Do Not Have a Birth Certificate

Order it from Texas Vital Records:

Texas Vital Statistics

Or contact the county where you were born.

Many nonprofits will help pay the fee.

4️⃣ You CAN Start Job Searching Without ID

You cannot complete hiring paperwork without proper ID, but you CAN:

  • Apply for jobs
  • Attend interviews
  • Explain your ID is being processed

What to say:

“My state ID is currently being processed. I have applied and expect it within a few weeks. I can provide documentation showing it is in progress.”

5️⃣ If Prison Never Submitted It

Call your unit’s:

  • Reentry office
  • Records department
  • Classification officer

Ask:

  • Was my ID application submitted?
  • When?
  • What address was used?

Sometimes it was mailed to the wrong address.

6️⃣ Help Paying for ID

If you cannot afford fees, contact:

  • Salvation Army
  • Catholic Charities
  • Local reentry programs
  • Texas Workforce Commission Reentry Services

Texas Workforce Commission

⚠ To legally complete employment paperwork (I-9), you must have:
  • Government-issued photo ID
  • Social Security card (or acceptable alternative)
Start the process immediately — do not wait on the prison.
